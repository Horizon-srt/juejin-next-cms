# 配置说明

可以提前学习一下strapi相关内容，教程比较简短

https://strapi.io/

需要环境配置

- nodejs v18.x
- Python (for sqlite)
- SQLite3

## Python

**For Ubuntu**

```
sudo apt-get update
sudo apt-get install python版本
```

**For MacOS**

建议安装Homebrew

```
brew install python3
```

## SQLite3

**For Ubuntu**

```
sudo apt-get install sqlite sqlite3
```

**For MacOS**

同样适用 Homebrew

```
brew install sqlite
```

上面两个安装过程仅供参考，网上相关内容很多。

## 运行

```
npm install
npm run develop
```
